# webinar-laravel

### Principais comandos para perfeita execução do push no projeto
- Primeiro, criei uma conta no github
- Criei um novo repositorio, mantendo ativado a opção de gerar o master, README.rm e o primeiro commit
- Apos, em meu terminal, fiz o clone do repositório:
```sh
git clone git@github.com:adrianodrix/webinar-laravel.git
```
- Em seguida, inclui este text no README.rm, e fiz o commit e push conforme segue:
```sh
git status
git commit -a -m "teste para push"
git push origin master
```
